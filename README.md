# Proyecto web: Mundial de Fútbol (para alumnos)

Material didáctico para que los estudiantes creen una página web con **HTML**, **CSS** y **JavaScript**.

## Contenido del repositorio

| Archivo | Descripción |
|---------|-------------|
| `INSTRUCCIONES-ALUMNOS.md` | Enunciado, requisitos y rúbrica de evaluación |
| `index.html` | Plantilla HTML con estructura y comentarios TODO |
| `css/estilos.css` | Estilos base personalizables |
| `js/script.js` | JavaScript funcional mínimo (toggle, formulario, bienvenida) |

## Cómo usar en clase

1. **Opción A — Desde cero:** Entrega solo `INSTRUCCIONES-ALUMNOS.md` y que creen la carpeta ellos mismos.
2. **Opción B — Con plantilla:** Reparte esta carpeta completa; los alumnos completan los TODO y personalizan diseño y contenido.
3. **Opción C — Demo:** Abre `index.html` en el navegador para mostrar el punto de partida antes de explicar HTML/CSS/JS.
