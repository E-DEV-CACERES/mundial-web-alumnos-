# Proyecto: Página web del Mundial de Fútbol

## Objetivo

Crear una página web sobre el **Mundial de Fútbol** usando solo **HTML**, **CSS** y **JavaScript** (sin frameworks).

## Entrega

- Carpeta del proyecto con: `index.html`, carpeta `css/` y carpeta `js/`
- La página debe abrirse correctamente al hacer doble clic en `index.html` o con Live Server

## Requisitos mínimos (obligatorios)

### HTML
- [ ] Estructura semántica: `header`, `nav`, `main`, `section`, `footer`
- [ ] Título principal sobre el Mundial
- [ ] Menú de navegación con al menos 3 enlaces internos (anclas)
- [ ] Sección **Historia** (breve texto sobre el torneo)
- [ ] Sección **Selecciones** (lista de al menos 4 países con bandera o emoji)
- [ ] Sección **Curiosidades** (lista o párrafos)
- [ ] Imagen relacionada con el fútbol (local o enlace)
- [ ] Tabla con datos (por ejemplo: año, sede, campeón)

### CSS
- [ ] Archivo externo `css/estilos.css` enlazado desde el HTML
- [ ] Paleta de colores coherente (verde, blanco, dorado, etc.)
- [ ] Tipografía legible
- [ ] Diseño **responsive** (que se vea bien en móvil y escritorio)
- [ ] Efecto hover en botones o enlaces del menú
- [ ] Estilos para la tabla (bordes, colores alternos en filas)

### JavaScript
- [ ] Archivo externo `js/script.js` enlazado al final del `body`
- [ ] Botón **"Mostrar/Ocultar curiosidades"** que muestre u oculte una sección
- [ ] Contador o mensaje de bienvenida al cargar la página (`DOMContentLoaded`)
- [ ] Validación simple: formulario de "¿Quién crees que ganará?" con alerta si el campo está vacío

## Requisitos extra (puntos adicionales)

- [ ] Modo oscuro con un botón para cambiar tema
- [ ] Galería de imágenes con CSS Grid o Flexbox
- [ ] Animación CSS (transición al hacer scroll o al pasar el mouse)
- [ ] Datos en un array JS y mostrarlos dinámicamente en la página
- [ ] Favicon personalizado

## Estructura de carpetas sugerida

```
mundial-web-alumnos/
├── index.html
├── css/
│   └── estilos.css
├── js/
│   └── script.js
├── img/          (opcional: imágenes propias)
└── INSTRUCCIONES-ALUMNOS.md
```

## Criterios de evaluación (100 puntos)

| Criterio | Puntos |
|----------|--------|
| HTML correcto y semántico | 25 |
| CSS: diseño, colores, responsive | 30 |
| JavaScript funcional | 25 |
| Creatividad y contenido | 10 |
| Organización del código | 10 |

## Consejos

1. Empieza completando el HTML con el contenido; después aplica estilos.
2. Usa comentarios en el código para explicar cada sección.
3. Prueba en el navegador después de cada cambio.
4. No copies páginas enteras de internet; escribe el texto con tus palabras.
5. Puedes usar emojis de banderas: 🇦🇷 🇧🇷 🇪🇸 🇫🇷 🇩🇪 etc.
